## 技术栈

- **静态生成**: [Hugo Extended](https://gohugo.io/) v0.163.2
- **主题**: [WebStack-Hugo](https://github.com/shenweiyan/WebStack-Hugo)
- **部署**: [GitHub Pages](https://pages.github.com/)
- **图标**: Font Awesome 5.15.4 + 自定义 SVG

## 本地开发

```bash
git clone https://github.com/meshbit/TelegramNavigation.git
cd TelegramNavigation/nav-hugo
git submodule update --init --recursive
hugo server -D        # 需安装 Hugo Extended
hugo --gc --minify    # 构建
```

## 贡献

欢迎 PR！数据文件 `nav-hugo/data/webstack.yml`：

```yaml
- taxonomy: 📦 资源频道
  icon: fas fa-box
  links:
  - title: 频道名称
    url: https://t.me/username
    description: 描述
    logo: https://example.com/icon.png  # 可选
```
